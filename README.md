# LeanMate

자료구조 과제 2용 OOP 기반 인터랙티브 웹앱입니다. 린매스업을 하는 사용자가 오늘 먹은 식단의 칼로리와 탄단지를 계산하고, 그룹원들과 운동 출석을 비교할 수 있도록 만들었습니다.

## 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 실행됩니다.

로컬 서버로 확인하려면 아래 명령을 사용할 수 있습니다.

```bash
python -m http.server 4173
```

이후 `http://127.0.0.1:4173/index.html`에 접속합니다.

## OOP 상속 구조

```text
FitnessRecord
├─ MealRecord
├─ WorkoutCheckIn
└─ GroupChallenge
```

- `FitnessRecord`: 공통 속성과 `run()` 메서드를 가진 부모 클래스
- `MealRecord`: 식단의 칼로리와 탄단지를 계산하도록 `run()` 오버라이딩
- `WorkoutCheckIn`: 사용자의 운동 출석을 갱신하도록 `run()` 오버라이딩
- `GroupChallenge`: 그룹원의 주간 출석 랭킹을 계산하도록 `run()` 오버라이딩

웹 화면에서 각 클래스 카드를 클릭하면 같은 `run()` 메서드가 객체 종류에 따라 서로 다른 결과를 출력합니다.

## 사용한 자료구조

- `Map`: 음식 이름을 key로 영양성분을 빠르게 조회
- `Array`: 오늘 식단 목록과 그룹원 목록 관리
- `Stack`: 마지막 식단 추가 기록 취소
- `Sort`: 주간 운동 출석 랭킹 계산

## 주요 기능

- 음식별 kcal, 탄수화물, 단백질, 지방 계산
- 하루 목표 kcal 대비 남은 섭취 가능 칼로리 표시
- 목표 단백질 달성률 계산
- 그룹원 운동 출석 체크
- 주간 출석 횟수 기반 랭킹
- OOP 상속 구조 시각화 및 다형성 인터랙션
